## Background
Originally conceived during an AI project requiring precise audio comparison, this tool was developed to bridge the gap between IT automation and professional Audio Engineering. With 10+ years of AV expertise from Akshay Jagtap, the "Lab" ensures that dubbed content meets industry standards.

## Key Features
- **Sync Score:** Phase correlation for lip-sync accuracy.
- **Clipping Detector:** Detects digital distortion in spiritual discourses.
- **Deep Metadata:** Extracts TBR, TBN, SAR, and DAR via FFmpeg.
- **Frequency Map:** Visual power shift analysis (Original vs. Dubbed).

## Installation
1. Install [FFmpeg](https://ffmpeg.org/) and add it to your PATH.
2. `pip install -r requirements.txt`
3. `python main.py`
